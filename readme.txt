No Nonsense Authors List provides the multi-author blog owner with a shortcode which can be used anywhere on their Wordpress blog to display a neat and responsive list of Authors.

== Description ==
Do you have multiple authors on your blog whom you want to acknowledge. No Nonsense Authors List provides a shortcode which can be used anywhere on their Wordpress blog to display a neat and responsive list of your Authors.
It comes already configured and it will nicely adapt to your website's theme.
The order of the list is randomized. Each item includes the author avatar, name, author bio and a link to read all the author posts.

== Installation ==
Simply install the plugin using the Add Plugin option and activate the plugin.
You are ready to directly use the shortcode [NO_NONSENSE_AUTHORS_LIST] anywhere in your website.

== Frequently Asked Questions ==
Q- Can I customize the list according to my liking?
A- Not in the present version. But we have planned to add a lot of options to this plugin in future.
